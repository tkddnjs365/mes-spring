# 애플리케이션 시작 과정
1. Spring Boot 애플리케이션 시작
2. @Configuration 클래스들 로딩 (SecurityConfig, WebConfig)
3. @Service, @Controller, @Component 등 빈 등록
4. 데이터베이스 연결 및 MyBatis 매퍼 스캔
5. ApplicationReadyEvent 발생 → SwaggerAutoOpenListener 실행
6. Swagger UI 자동으로 브라우저에서 열림
7. 서버 대기 상태 (포트 7010)

# 로그인 API 호출 과정 ( SuperUser Login 기준 )
[클라이언트] → [Controller] → [Service] → [Mapper] → [Database]
     ↓              ↓            ↓          ↓          ↓
1. POST /api/superUser/login 요청
2. @Valid로 RequestDto 검증
3. Service.login() 호출
4. Mapper.superUserLogin() 호출
5. SQL 쿼리 실행
     ↑              ↑            ↑          ↑          ↑
[Response] ← [ResponseDto] ← [SuperUserDto] ← [ResultSet] ← [SELECT 결과]

상세 실행 순서

요청 수신: 클라이언트가 POST /api/superUser/login으로 JSON 요청
데이터 검증: @Valid가 RequestSuperUserDto의 @NotBlank 검증 수행
컨트롤러 처리: userId 공백 제거 후 Map에 파라미터 저장
서비스 호출: SuperUserService.login() 메서드 실행
데이터 접근: SuperUserMapper.superUserLogin() 호출
SQL 실행: MyBatis가 XML의 SELECT 쿼리 실행
결과 매핑: 조회 결과를 SuperUserDto 객체로 변환
응답 처리:

사용자 정보가 있으면 → ResponseSuperUserDto 생성 후 200 응답
사용자 정보가 없으면 → 401 Unauthorized 응답
